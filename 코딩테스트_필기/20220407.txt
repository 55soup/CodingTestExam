문자열
- char형 변수 : 문자 1개의 값을 저장
- 컴퓨터에서 문자는 사실상 정수이다. (코드값으로 저장하기 때문에)
- "숫자 "는 수를 표현하기 위한 "문자"이다.
- ASCII코드 : 영문자, 숫자, 기호들의 번호를 정의하고 1byte로 처리
- 1byte 표현할 수 있는 범위: `128 ~ 127
- "0" : 숫자로 컴퓨터 내에서 정수 48로 저장된다.
따라서 숫자 '0'을 정수로 바꾸려면 '0'-48의 뺄셈을 해야한다.
- 숫자 0 => 정수변환 : '0'(char 0의 아스키코드는 48) -48
  char c = '9';
  int n = c - 48;  //n에 정수 9가 배정된다.

숫자
수를 표현하기 위한 문자 : 1,2,one, two, 하나, 둘...

수
컴퓨터가 처리하는 수
-숫자 '0'은 컴퓨터 내부에서 정수 48로 저장 => 숫자 '0'을 정수로 바꿀경우
'0'-48이다.
- 같은 예로 '9'는 57이므로 -48을 해야 정수가 된다.

문자열의 이해
- 배열의 길이와 배열에 저장된 길이는 다르다 
=> 문자열의 끝을 표시하기 위해 문자열의 마지막에 정수 0을 붙인다.
- 널(null)문자 : 문자열의 마지막을 표시하는 문자, \n

문자열 처리 규칙
- ASCII에 정의된 문자를 제외한 나머지는 모두 문자열로 처리함.
- 문자열 처리 함수는 문자열의 시작주소( 첫 글자의 주소)를 - 포인터 사용

1차원 문자열 배열의 이해
char s1[] = "COS PRO"; // 크기를 알아서 배정
char s2[10] = {'k','o','r','e','a'};  //크기 수동배정 문자열이 넣어지지않는 공간은 쓰레기값이 들어감

★문자열 찍을 때 for문 사용해야함. (%s로 출력이 되긴함)

2차원 문자열 배열의 이해
Char color[5][10] = {"red", "green", "blue", "brown", "cyan"};
위의 2차원 배열을 단어 단위로 출력하고, 문자 단위로 출력해보자.

문자열의 길이 알아내기
- 문자열의 길이는 널문자 전까지의 문자 개수이며 배열의 길이보다 작다.
- **널문자는 정수 0또는 문자'\0'로 작성한다. ( null은 숫자로0, 문자로\0)
